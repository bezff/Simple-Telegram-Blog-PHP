# Simple Telegram Blog (PHP)

Легкий движок блога на чистом PHP, контент в который публикуется через Telegram-бота.
Идеально подходит для личных заметок, фотоблога или портфолио.

## Особенности
- **Без базы данных SQL** (используется JSON).
- **Чистый PHP** .
- **Telegram Bot API** для управления контентом.
- **Поддержка фото и текста**.
- **Простая авторизация** по Telegram ID.

## Установка

### Шаг 1: Создание бота
1. Откройте в Telegram бота **@BotFather**.
2. Отправьте команду `/newbot`.
3. Придумайте имя и юзернейм для бота.
4. Скопируйте полученный **API TOKEN**.

### Шаг 2: Получение вашего ID
1. Откройте в Telegram бота **@userinfobot**.
2. Отправьте ему любое сообщение.
3. Скопируйте число из поля `Id`. Это ваш `ADMIN_ID`.

### Шаг 3: Настройка кода
1. Клонируйте репозиторий.
2. Откройте файл `config.php`.
3. Вставьте свои `BOT_TOKEN` и `ADMIN_ID`.
4. Укажите `BASE_URL` — это ссылка на ваш сайт (например, `https://mysite.com/webhook.php`).

### Шаг 4: Загрузка на сервер
1. Загрузите файлы на хостинг.
2. **Важно:** Установите права на запись (`755` или `777`) для папок:
   - `data/`
   - `assets/uploads/`
   Если этого не сделать, бот не сможет сохранять посты и картинки.

### Шаг 5: Установка Webhook (Связь бота с сайтом)
Чтобы бот знал, куда отправлять данные, нужно открыть в браузере специальную ссылку:

https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ВАШ_САЙТ.com/webhook.php

*Замените `ВАШ_ТОКЕН` и `ВАШ_САЙТ.com` на свои данные.*

Если вы увидите ответ `{"ok":true, "result":true, ...}`, значит всё настроено верно!

## Использование
1. Зайдите в своего бота.
2. Нажмите `/start` (вы должны получить приветствие админа).
3. Отправьте текст или фото с подписью.
4. Проверьте главную страницу сайта.
